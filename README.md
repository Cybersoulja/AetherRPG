# Aethermoor Chronicles

Aethermoor Chronicles is an interactive text-based RPG adventure powered by Ink.js and Taskade AI.

## Features

*   **Interactive Fiction**: Experience a rich, branching narrative powered by the Ink.js storytelling engine.
*   **AI Dungeon Master**: Dynamic responses from Taskade AI agents acting as your personal DM.
*   **Living NPCs**: Meet intelligent NPCs and allies powered by advanced AI personalities.
*   **Character Progression**: Level up your character with meaningful choices and stat improvements.

## Getting Started

### Prerequisites

*   Node.js and npm installed.

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/your-username/aethermoor-chronicles.git
    ```
2.  Install the dependencies:
    ```bash
    npm install
    ```

### Running the Application

1.  Start the development server:
    ```bash
    npm run dev
    ```
2.  Open your browser and navigate to `http://localhost:5000`.

## Project Structure

*   `client/`: Contains the frontend code, built with React and TypeScript.
*   `server/`: Contains the backend code, built with Express and TypeScript.
*   `shared/`: Contains shared code, such as the database schema.
